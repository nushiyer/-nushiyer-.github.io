HTML Reference Page

The HTML element (<html>, </html>)

This element defines the entire document and is placed after !DOCTYPE tag. Here is an example:

<!DOCTYPE html>

<html>

The Head element (<head>, </head>)

This element is a container for all the head elements and is placed after the HTML tag.

This is the title of the Page

The Title element (<title>, </title>)

This element defines the title of any document and is placed after the head tag.

This is the title

The Body element (<body>, </body>)

This element defines the bosy of the document and is placed after the HTML tag.

This is the body

The Headings element (<h1-h6>, </h1-/h6>)

This element defines the headings set for a page and is placed within the body sytax tags.

This is an example of the slightly less important heading

The Paragraph element (<p>, </p>)

This element defines the paragraph features found within the body syntax tags.

For example: This is a paragraph

The Breaks element (<br>)

This element defines breaks in between paragraphs and other information and is found within the body syntax tags.

The Lines element (<hr>)

This element defines a method of separating two pieces of information from each other via a line and is found within the body syntax tags.

This paragraph is separate from

this paragraph.

The Comments element (<!-- -->)

This element defines the access to comments on a page and is normally found before or after a paragraph.

Here is the paragraoh I am commenting on.

The Links element (<a>, </a>)

This element defines the link from on page to another and is found after a paragraph referencing or directing to another source of information.

This is information about computer science

The Images element (<img>, </img>)

This element defines images used on sites and can be found anywhere within the body syntax tags

The Lists element (<ul, ol, li>, </ul, /ol, /li>)

This element defines the lists attribute used to organize a series of information and can be found anywhere within the body syntax tags.

  3. Avocado
